JavaScript loops

JavaScript for loop Syntax

  • initialExpression - Initializes a counter variable.
  • condition - The condition to be evaluated. If true, the body of the for loop is executed.
  • updateExpression - Updates the value of initialExpression.
Once an iteration of the loop is completed, the condition is evaluated again. The process continues until the condition is false.
                        
                        
                            while (condition) {
                                // body of loop
                            }

Iterate Through an Array

                
                    const fruits = ["apple", "banana", "cherry"];
                    for (let i = 0; i < fruits.length; i++) {
                        console.log(fruits[i]);
                    }

JavaScript while Loop

  • The while loop first evaluates the condition inside ( ).
  • If the condition evaluates to true, the code inside { } is executed.
  • Then, the condition is evaluated again.
  • This process continues as long as the condition evaluates to true.
  • If the condition evaluates to false, the loop stops.
                        
                        
                            for (initialExpression; condition; updateExpression) {
                                // for loop body
                            }

Iterate Through an Array

                
                    // initialize variable i
                    let i = 1;

                    // loop runs until i is less than 4
                    while (i < 4) {
                        console.log(i);
                        i += 1;
                    }

JavaScript do...while Loop

  • Then, it evaluates the condition inside ( ).
  • If the condition evaluates to true, the code inside { } is executed again.

Iterate Through an Array

                
                    let i = 3;

                    // do...while loop
                    do {
                        console.log(i);
                        i--;
                    } while (i > 0);

JavaScript for... of Loop

  • iterable - an iterable object (array, set, strings, etc).
  • element - items in the iterable
                        
                        
                            for (element of iterable) {
                                // body of for...of
                            }
                        
                    
                
                    // array
                    const students = ['John', 'Sara', 'Jack'];
                    
                    // using for...of
                    for ( let element of students ) {
                    
                        // display the values
                        console.log(element);
                    }

JavaScript for...in loop

  • object - The object whose keys we want to iterate over.
  • key - A variable that stores a single key belonging to object.
                        
                        
                            for (key in object) {
                                // body of for...in
                            };
                        
                    
                
                    const salaries = {
                        Jack: 24000,
                        Paul: 34000,
                        Monica: 55000
                    };
                    
                    // use for...in to loop through
                    // properties of salaries
                    for (let i in salaries) {
                    
                        // access object key using [ ]
                        // add a $ symbol before the key
                        let salary = "$" + salaries[i];
                    
                        // display the values
                        console.log(`${i}: ${salary}`);
                    };

JavaScript forEach()

  • function(currentValue, index, arr) - a function to be run for each element of an array
  • currentValue - the value of an array
  • index (optional) - the index of the current element
                        
                        
                            array.forEach(function(currentValue, index, arr))
forEach with Arrays
                
                    let students = ['John', 'Sara', 'Jack'];

                    // using forEach
                    students.forEach(myFunction);
                    
                    function myFunction(item) {
                    
                        console.log(item);
                    }
forEach with Arrow Function
                
                    // with arrow function and callback

                    const students = ['John', 'Sara', 'Jack'];
                    
                    students.forEach(element => {
                      console.log(element);
                    });
